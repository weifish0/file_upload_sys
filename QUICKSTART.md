# 快速開始指南

## 一鍵啟動

在專案目錄下執行：

```bash
./start.sh
```

這個腳本會自動：
- ✅ 建立虛擬環境（如果不存在）
- ✅ 安裝所需依賴（如果尚未安裝）
- ✅ 初始化資料庫（如果不存在）
- ✅ 啟動應用程式

## 訪問應用

### 家長表單（無需登入）
🔗 **http://localhost:5002**

在這裡家長可以：
- 填寫孩子姓名
- 填寫聯絡資訊（選填）
- 上傳檔案（最大 10MB）

### 管理後台
🔗 **http://localhost:5002/admin/login**

預設管理員帳號：
- **使用者名稱**: `admin`
- **密碼**: `admin123`

⚠️ **重要**：首次登入後請立即更改密碼！

## 管理後台功能

登入後您可以：

### 📊 查看統計資訊
- 總提交數
- 總檔案大小
- 即時數據統計

### 🔍 搜尋與篩選
- 按孩子姓名搜尋
- 按家長資訊搜尋
- 按檔案名稱搜尋

### 📥 下載檔案
- 點擊「下載」按鈕下載個別檔案
- 使用「匯出 CSV」匯出所有記錄

### 🗑️ 刪除記錄
- 刪除不需要的提交記錄
- 同時刪除檔案和資料庫記錄

## 檔案格式支援

系統支援以下檔案格式：

- 📄 **文件**：PDF, DOC, DOCX, TXT
- 📊 **試算表**：XLS, XLSX
- 📽️ **簡報**：PPT, PPTX
- 🖼️ **圖片**：JPG, JPEG, PNG, GIF
- 📦 **壓縮檔**：ZIP, RAR

## 常見問題

### Q: 如何停止伺服器？
**A**: 在終端機中按 `Ctrl+C`

### Q: 如何更改管理員密碼？
**A**: 目前需要直接修改資料庫。在未來版本會添加密碼修改功能。

### Q: 檔案儲存在哪裡？
**A**: 所有上傳的檔案都儲存在 `uploads/` 目錄中。

### Q: 如何備份資料？
**A**: 
1. 備份 `workshop.db` 資料庫檔案
2. 備份 `uploads/` 目錄中的所有檔案

### Q: 如何重置系統？
**A**:
```bash
rm workshop.db
rm -rf uploads/*
python init_db.py
```

### Q: 埠口 5002 被佔用怎麼辦？
**A**: 修改 `app.py` 最後一行的 `port=5002` 為其他埠號，例如 `port=5002`

## 安全建議

在正式環境中使用前，請務必：

1. ✅ 更改管理員密碼
2. ✅ 修改 `config.py` 中的 `SECRET_KEY`
3. ✅ 設定適當的檔案大小和類型限制
4. ✅ 使用 HTTPS
5. ✅ 定期備份資料

## 技術支援

如遇到任何問題，請檢查：
1. Python 版本（建議 3.8+）
2. 所有依賴是否正確安裝
3. 埠口是否被佔用
4. 檔案權限是否正確
